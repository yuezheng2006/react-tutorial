# React-Tutorial

这是一个刚从 Vue 转向 React 技术栈的小白学习教程。

## 创建应用程序

使用 [Create React App](https://www.html.cn/create-react-app/docs/getting-started/) 创建 React 单页应用（SPA）：

```javascript
npx create-react-app react-tutorial
```
